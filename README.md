# Foco

Clon funcional de un temporizador Pomodoro, construido con **Codex** para el Proyecto 8 de Platzi.

## Enlaces públicos

- 🚀 **Aplicación:** [foco-pomodoro.jpalomera.chatgpt.site](https://foco-pomodoro.jpalomera.chatgpt.site/)
- 🎥 **Video explicativo:** [youtu.be/MF2DKVoKYyw](https://youtu.be/MF2DKVoKYyw)
- 💻 **Repositorio:** [github.com/jpalomera/foco-pomodoro](https://github.com/jpalomera/foco-pomodoro)

## Funciones

- Temporizador de enfoque, descanso corto y descanso largo.
- Lista de tareas guardada localmente.
- Estadísticas diarias y actividad de los últimos siete días.
- Duraciones personalizables, sonido y notificaciones.
- **Mejora propia:** ritmo adaptable, que recomienda sesiones más cortas o largas según cuántas completas o saltas.

## Ejecutar localmente

Abre `index.html` directamente en un navegador. Para notificaciones completas:

```bash
npx serve .
```
